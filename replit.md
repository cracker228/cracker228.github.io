# Telegram Bot

A Node.js Telegram bot using Telegraf framework.

## Setup

1. Get your bot token from [@BotFather](https://t.me/botfather) on Telegram
2. Add your `BOT_TOKEN` to the Secrets tab
3. Run the bot

## Structure

- `bot.js` - Main bot file with handlers
- `package.json` - Dependencies

## Commands

- `/start` - Welcome message
- `/help` - Help information

## Customization

Edit `bot.js` to add your own commands and handlers.
